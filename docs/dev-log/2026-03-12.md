  - disabled proxy inheritance for Playwright browser runs
  - added `--no-proxy-server` launch arg

### Product issues found and fixed by the browser run

1. `/apps` crashed after login because DB-backed `workspace_apps.tags` were returned as JSON strings instead of `string[]`.
   - fixed in `apps/gateway/src/services/persistent-workspace-service.ts`
   - added defensive normalization in `apps/web/src/lib/apps-client.ts`
   - extended tests so the workspace contract now locks `tags` as arrays

2. login could race with background SSO discovery when users immediately typed a password and submitted.
   - fixed in `apps/web/src/app/(auth)/login/page.tsx`
   - once a password is present, SSO discovery is cancelled/skipped for that input state

3. browser E2E assertions were aligned to the current UI contract:
   - generic password-policy error text on `/register`
   - stable SSO and profile/security selectors
   - current admin placeholder headings (`Users`, `Groups`, `Apps`, `Audit`)
